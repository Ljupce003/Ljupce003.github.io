Algorithm Visualizer

An interactive web app that visually demonstrates how common sorting and pathfinding algorithms work step-by-step. Supports Bubble Sort, Merge Sort, Quick Sort, and Dijkstra's algorithm. Useful as an educational tool for understanding algorithmic complexity.
